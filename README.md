# AI Text Summarizer

AI-Based Text Summarization Platform built using React.js, Node.js and HuggingFace API.

## Features

- Text summarization
- PDF summarization
- AI-generated summaries
- File upload support
- Clean UI

## Technologies Used

Frontend:
- React.js
- CSS
- Axios

Backend:
- Node.js
- Express.js

AI:
- HuggingFace Transformers API

## Installation

### Frontend

```bash
cd client
npm install
npm run dev
```

### Backend

```bash
cd server
npm install
npm run dev
```

## Project Workflow

User Input → Backend API → AI Model → Summary Output
